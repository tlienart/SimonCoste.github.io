I find this method computational and I never remember the block-inversion formula (6).

Instead, there is a simpler, more conceptual path: observe that log⁡f(x,y)\log f(x,y)logf(x,y) is a quadratic function in (x,y)(x,y)(x,y), hence when yyy is fixed, log⁡f(x,y)\log f(x,y)logf(x,y) is still a quadratic function in xxx.
