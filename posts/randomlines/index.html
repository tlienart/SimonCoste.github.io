Random line on the plane

August 2021

How do you draw lines at random in the plane, in such a way that the corresponding process does not depend on the origin?

Representing lines with points

Let ℓ\ellℓ be 2d line, ie a one-dimensional subspace of R2\mathbb{R}^2R2.

Mathematically, it can be represented in many ways: for instance, if it is not vertical, there are two unique numbers α,β\alpha, \betaα,β such that ℓ={(x,αx+β):x∈R}\ell = \{ (x, \alpha x + \beta) : x \in \mathbb{R} \}ℓ={(x,αx+β):x∈R}. Random lines

A random line process is simply the set of lines obtained by sampling a random point process on the strip A\mathscr{A}A and looking at the lines they represent.

For instance, a Poisson line process is nothing but the set of lines obtained by sampling a Poisson point process on A\mathscr{A}A.

One could easily create many kinds of random processes by first sampling classical point process Φ\PhiΦ in A\mathscr{A}A, and then using the correspondance to get the line analog of this process. But...

Is it shift-invariant ?

Just as points in Rd\mathbb{R}^dRd, lines can be shifted. If xxx is a point and ℓ\ellℓ a line, ℓ+x\ell + xℓ+x is simply the set {y+x:y∈ℓ}\{y+x : y \in \ell \}{y+x:y∈ℓ}. A crucial question in geometric probability is to create random line processes LLL that are shift-invariant: the distribution of LLL, and the distribution of L−x={ℓ−x:ℓ∈L}L - x = \{\ell - x : \ell \in L \}L−x={ℓ−x:ℓ∈L} must be the same.

While such a condition is not too difficult to check for many interesting point processes[1], it is not so obvious for the kind of line processes I described above.

Indeed, when a line ℓ\ellℓ represented by (θ,r)(\theta, r)(θ,r) is shifted by xxx, its representation in A\mathscr{A}A undergoes a non trivial transformation: if α\alphaα is the argument of xxx, then

ℓ−x↔(θ′,∣r′∣) \ell - x \quad \leftrightarrow \quad (\theta', |r'|)ℓ−x↔(θ′,∣r′∣)

where r′=r+∣x∣cos⁡(θ−ρ)r' = r + |x|\cos(\theta - \rho)r′=r+∣x∣cos(θ−ρ), and θ′=θ\theta' = \thetaθ′=θ if r>0r>0r>0, otherwise it is equal to (θ+π)(\theta + \pi)(θ+π) modulo 2π2\pi2π.

Hence, if we note Tx:A→AT_x : \mathscr{A} \to \mathscr{A}Tx​:A→A the group of transformations described above, we obtain the simple caracterization: a line process is shift-invariant if its representation in A\mathscr{A}A is TxT_xTx​-invariant for all xxx. While this can be checked for what I described above as the Poisson line process, it also prevents other naive constructions from being stationary. Is it isotropic ?

The same question can be asked with isotropy, that is: rotation invariance. When a line ℓ\ellℓ is represented by (θ,r)(\theta, r)(θ,r) in A\mathscr{A}A, rotating around the origin by an angle τ\tauτ is the same thing as replacing θ\thetaθ by θ+τ\theta + \tauθ+τ (modulo 2π2\pi2π). Consequently, a line process is isotropic if and only if its representation in A\mathscr{A}A is invariant with respect to the transformations Rτ: (θ,r)→(θ+τ,r)R_\tau : (\theta, r) \to (\theta + \tau, r)Rτ​: (θ,r)→(θ+τ,r) where the angle addition is taken modulo 2π2\pi2π.

A typical example of non-isotropic point process is obtained by choosing a finite set of directions, say (θ1,…,θk)(\theta_1, \dotsc, \theta_k)(θ1​,…,θk​), and for each of them to sample a translation-invariant point process Φj\Phi_jΦj​ on R\mathbb{R}R. The union

{(θi,x):i∈[k],x∈Φi}\{ (\theta_i, x) : i \in [k], x \in \Phi_i \}{(θi​,x):i∈[k],x∈Φi​}

is translation-invariant, but certainly not isotropic.
